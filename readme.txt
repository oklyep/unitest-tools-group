Поддержка массовых операций и общего доступа к контейнерами из образа test-tools

1. Предоставление информации и контейнерах из этого образа на этом докер-хосте и их параметрах
2. Ограничение на одновременный запуск
3. Автоматическая остановка после N часов работы
4. Асинхронный бэкап баз данных и обновление на свежие сборки для целевых контейнеров.
Опрерации производятся через очередь (чтобы не весить сервера баз),
количество очередей равно количеству серверов баз данных , используемых целевой группой контейнеров
